# Bootcamp 2021: AI and AIoT Serverless Multi-Tenant SaaS APIs
* Technologies which are going to be covered during this bootcamp
    * Development Languages
        * Python
        * Rust
        * TypeScript
    * TensorFlow as ML framework
    * AWS as Serverless Cloud platform
    * AWS CDK with TypeScript for Infrastructure as Code.
